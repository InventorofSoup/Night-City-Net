# Trauma Team International

An in-world, phone-friendly static website for a Cyberpunk RED campaign.

## Use locally

Open index.html in a browser. It does not need a server or installation.

## Publish with GitHub Pages

1. Create a GitHub repository, such as night-city-net.
2. Add this trauma-team folder to it.
3. In Settings > Pages, choose Deploy from a branch, select main, then select the repository root.
4. The site will publish at https://YOUR-USERNAME.github.io/REPOSITORY-NAME/trauma-team/.

The quote calculator and sign-in screen are fictional and run only in the visitor’s browser; they do not store information.
